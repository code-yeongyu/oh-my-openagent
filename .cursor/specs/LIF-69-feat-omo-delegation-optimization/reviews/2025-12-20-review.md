## ⚠️ Warnings (Recommendations)

### 1. Missing `*.mdx` Pattern (Medium)

**Location**: `src/hooks/governance-docs-delegation/types.ts`

The `DOCS_PATH_PATTERNS` array is missing `*.mdx` which is specified in FR-002.

**Current**:
```typescript
export const DOCS_PATH_PATTERNS: string[] = [
  "docs/",
  "README.md",
  "README.*.md",
  "CHANGELOG.md",
  "changelog/",
  ".cursor/memory/",
  "context/memory/",
]
```

**Recommended**:
```typescript
export const DOCS_PATH_PATTERNS: string[] = [
  "docs/",
  "*.md",      // Cover all markdown
  "*.mdx",     // Cover MDX files
  "README.md",
  "README.*.md",
  "CHANGELOG.md",
  "changelog/",
  ".cursor/memory/",
  "context/memory/",
]
```

### 2. Pattern Duplication (Low)

Both `governance-docs-delegation/types.ts` and `shared/delegation-policy.ts` define docs path patterns. Consider consolidating to a single source of truth.

### 3. Intent Detection Deferred (Tracking)

FR-003 (intent-based triggers with doc verbs/nouns) is implemented in `delegation-policy.ts` but not wired into the hook. This is correctly deferred to P1 per tasks.md (T031-T032).
