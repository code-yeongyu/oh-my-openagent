# Try Hard

## Overview

Utilize planning instructions and tools extensively to create an extensive and highly detailed plan and list of to-do tasks. Every task should be single-purpose, with all tasks completing the request from start to finish.

## User Input

```text
$ARGUMENTS
```

## Approach

1. **First-Principles Thinking**
   - Break down the problem to its fundamental components
   - Question all assumptions
   - Build up from basic truths

2. **Chain-of-Thought Reasoning**
   - Think through every step logically
   - Document reasoning at each decision point
   - Evaluate alternatives before choosing

3. **Comprehensive Planning**
   - Create single-purpose tasks
   - Ensure tasks flow logically from start to finish
   - Include all necessary steps, no matter how small

4. **Decision Evaluation**
   - Evaluate decisions for highest odds of success
   - Consider risks and mitigations
   - Document trade-offs

5. **Tool Utilization**
   - Use all available tools to their full extent
   - Gather necessary context and information
   - Validate assumptions with data

## Output

A comprehensive plan with:
- Detailed task breakdown
- Clear dependencies
- Risk assessment
- Success criteria
- Validation steps
