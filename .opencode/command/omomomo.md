## What is Matrixx?

**Matrixx** is a batteries-included OpenCode plugin that turns your AI coding agent into a coordinated team of specialists:

- 🤖 **12 Specialized Agents**: Morpheus (orchestrator), Oracle (strategy), Keymaker (deep work), Cipher (DSL/parsers), Sentinel (security), Trinity (fast grep), and more
- 🔧 **25+ Developer Tools**: Full LSP capabilities, AST-Grep search/replace, session management, tmux integration
- 🔄 **Parallel Background Agents**: 5+ running simultaneously with circuit breakers so they don't burn tokens forever
- 🎯 **41 Lifecycle Hooks**: Todo enforcement, think-mode escalation, context injection, error recovery
- 📚 **Built-in MCPs**: Exa (web search), Context7 (official docs), Grep.app (GitHub code search)
- 🛡️ **Security by Default**: Secret leak guard, env file protection, Sentinel auditing agent
