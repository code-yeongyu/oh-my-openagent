# Quickstart Guide

This guide will help you set up {{project.name}} and run your first example.

## Prerequisites

Before you begin, ensure you have:

    Version 18.0 or higher
  </Card>
  <Card title="Package Manager" icon="box">
    npm, yarn, pnpm, or bun

<Tabs>

## Configuration

Create a configuration file in your project root:

```bash
touch {{project.config_file}}
```

Add your configuration:

```json
{
  "name": "my-project",
  "version": "1.0.0"
}
```

```bash

```

You should see the version number printed to the console.

## Troubleshooting

<AccordionGroup>
  <Accordion title="Installation fails with permission error">
    Try running with elevated permissions or use a Node version manager like nvm.
    
    ```bash
    nvm use 18
    npm install {{project.name}}
    ```
  </Accordion>
  <Accordion title="Module not found error">
    Ensure your `package.json` has the correct dependencies and run:
    
    ```bash
    rm -rf node_modules
    npm install
    ```
  </Accordion>
  <Accordion title="Configuration not loading">
    Verify your configuration file is in the project root and has valid JSON syntax.
  </Accordion>
</AccordionGroup>
